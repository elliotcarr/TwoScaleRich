#######################################################################
#                           TwoScaleRich
#######################################################################

A computational code for simulating unsaturated water flow in a double 
porosity medium using a two-scale modelling strategy based on Richards' 
equation.

The code uses a control volume finite element method on unstructured 
two-dimensional meshes at both scales.

This code is experimental, still in a very early development stage and 
changing rapidly, so don't use it for anything important.

Written by Elliot Carr.
